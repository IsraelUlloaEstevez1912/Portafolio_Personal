Control de Versiones

En esta parte hicimos uso de Git y GitHub, herramientas esenciales para el control de versiones en el desarrollo de software. Git es un sistema de control de versiones distribuido que permite a los desarrolladores rastrear cambios en su código, colaborar con otros y gestionar diferentes versiones de un proyecto. GitHub es una plataforma basada en la nube que facilita la colaboración y el alojamiento de repositorios Git, permitiendo a los desarrolladores compartir su código, revisar cambios y trabajar juntos en proyectos de manera eficiente.
